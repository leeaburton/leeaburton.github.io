Post: Website

Feb 19, 2018 2 min read

I’m always forgetting how to code. It’s not a great trait for someone who’s entire career depends on it. As i said on my homepage I want this site to be a place to keep the kind of snippets I always forget. So to begin I’ll put here the info relevant for the site itself.

Firstly, this site is built by Hugo and not Jekyll. I installed Hugo with Homebrew, which is a great package manager for mac computers.

I write the files on my computer, which are easy to read, modify and create. Then Hugo compiles them for me into a ‘public’ folder.

These changes can be viewed locally before i make anything public, so I can check I didn’t do too much damage!

This can be done by typing

hugo server

and then opening localhost:1313 in my web browser

The site’s progress is easily documented, tracked and kept transparent by github. Github also provides the domain, as you can see from the address bar.

After I use Hugo to build the files, I go to the public directory, add the changes, comment on the changes and push to the repository.

The commands for this are:

hugo

cd public

git add .

git commit -m “explanation of most recent changes”

git push

Then it takes about 10 minutes before the site updates to the most recent version.
